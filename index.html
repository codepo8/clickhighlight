Click highlighting with CSS transitions

Click on this page and goo "wheeeee" when you see the growing dots.

If you want to customise, just send your settings to init():

clickhighlight.init({
  size:        '300px', // the maximum size of the dot
  duration:    '2s',    // duration of the effect (seconds) 
  colour:      'green', // the dot colour - RGBA = transparency 
  nohighlight: 'notme'  // class of elements not highlighted
});
